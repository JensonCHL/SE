# React + TypeScript + Vite

Step to use website

1. in terminal write: git clone https://github.com/JensonCHL/SE 
2. cd to file
3. npm i (to install necessary package)
4. cd to server
5. npm start (to start server)
6. cd back
7. npm run dev (to start the front end)
